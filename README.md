# Community Hero

Community Hero is a smart civic issue reporting and management platform where **Citizens**, **Volunteers**, and **Authorities** work together to report, verify, and resolve public issues efficiently.

## 🚀 Features

- Citizen, Volunteer & Authority Login
- Report civic issues with photos
- AI-based issue severity analysis
- Interactive OpenStreetMap integration
- Complaint tracking and status updates
- Volunteer verification workflow
- Authority management dashboard
- Firebase Authentication & Firestore
- Responsive modern UI

---

## 📋 Prerequisites

- Node.js (v18 or later)
- npm

---

## ⚙️ Installation

```bash
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root and add:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## ▶️ Run the Project

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
